# emu Razer Kishi to x360
HID => x360 xinput for Razer Kishi (Android Gamevice)
使用[ViGEm](https://github.com/nefarius/ViGEmBus)和[HidSharp](https://github.com/IntergatedCircuits/HidSharp)两个美妙的工具让这个奇怪的安卓串流伸缩手柄得以使用USB全功能连接到Windows使用...
~~其实原本也就俩线性扳机会被识别为按钮~~
你需要先安装[ViGEm](https://github.com/nefarius/ViGEmBus/releases)驱动，然后再下载运行[这个软件](https://github.com/Kissycat/RazerKishi2x360/releases/download/v1/kishi.exe)。
